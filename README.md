# 🔥 React Native Amazon-Clone App

## ✨ Features :

- 🔥 Features:
- 🔄 Redux for state management.
- 📦 Local Storage (AsyncStorage).
- 🌐 Integration with a DummyJson API.
- 🚀 Smooth navigation with React Navigation.
- 🔍 Search functionality to enhance user experience.
